Projects

  1. RFID Based Employee attendance register

    • Developed using LPC2129 Arm7 microcontroller. Rfid module taking the card details and added with other nec-essary details acquired from arm7 microcontroller, then send to computer and saving it in the file with name, date, time and entry or exit status.
    • The protocols used are UART and I2C

  2. IOT Based Automated Medicine Dispenser

    • Developed using ESP32 Microcontroller.
    • Anyone with a registered RFID tag can access the dispensers.
    • Users need to download the mobile app to obtain medicines and check the dispenser's medicine levels by entering 4-digit PIN for verification.

  3. Smart Railway Gate Opening System using IoT

    • Which present an automatic railway gate control system using RFID and ESP32.
    • This system aims to reduce railway accidents by automating the gate control process.
    • It uses a two-step verification: first, the loco pilot presses a button to signal the train's approach, and then an RFID tag is scanned by a reader near the level crossing.
    • Only after both steps will the gates open or close.
